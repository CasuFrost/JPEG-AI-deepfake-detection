## Forensic analysis
Risulta di cruciale importanza osservare come JPEG AI modifica le componenti frequenziali di una data immagine, che sia reale oppure un deepfake. Indipendentemente dalla veridicità di un immagine, JPEG AI tende a lasciare delle tracce riconoscibili nelle immagini nel dominio della frequenza, ed è anche possibile riconoscere come tali tracce varino in base al BPP.
